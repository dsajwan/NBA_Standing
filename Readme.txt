This project is used to determine team standing of NBA 2015-2016 based on previous two season data.


Data mining technique used in prediction
Classification
Random forest
Decision Tree

Python programming language has been used along with packages such as numpy,pandas and sklearn.

Anaconda is used as IDE for python.

Code_NBA.txt contains the python code

NBA_STANDING.docx contains the detail decsription of project and its findings.
